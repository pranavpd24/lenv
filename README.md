# LENV - Linux Environments for Developers

Project-scoped Linux environments with venv-like UX for Windows developers.

## Installation
```bash
pip install lenv
```

## Quick Start
```bash
cd your-project
lenv init
lenv activate
pip install airllm  # Linux pip!
python train.py     # Linux Python!
exit
```
